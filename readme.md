# Webdriverio + Appium + Android testing
--------------------------------------------
## Setup & dependencies
- Java JDK
- Node & npm
- Android studio & emulator
- VS code 
- Appium server
- Appium example app 

### Java JDK
Check version of java:    
```java -version```

Download link for Java: https://www.oracle.com/java/technologies/javase-downloads.html

### Node JS
Check version of node:    
````node -v````

Check npm version:   
```npm -v```

Download link for node: https://nodejs.org/en/

### Android Studio:
Download link: https://developer.android.com/studio/install
1) Configure -> AVD manager
2) Create virtual device
3) Chose wanted phone from a list (Pixel 2 is most standard with Full HD)
4) Chose Android version - download most recent 
5) Name your device 
6) Start device with play button

### Appium
Download appium from link:
https://github.com/appium/appium-desktop/releases/tag/v1.15.1 (versions above 1.15.1 wont work)
Follow installation of appium and after installation start server on port 4723 (default appium port)
